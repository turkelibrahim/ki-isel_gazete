# E-Gazete Sürümü — Kavram Ağı Kaldırıldı

Bu sürümde kullanıcı isteğiyle `Kavram Ağı / Knowledge Graph` özelliği tamamen kaldırıldı.

## Kaldırılanlar
- Üst menüdeki `Kavram Ağı` butonu kaldırıldı.
- `Sana En Uygun Haberler` başlığındaki `Kavram Ağı` butonu kaldırıldı.
- Ana akış başlığındaki `Kavram Ağı` butonu kaldırıldı.
- Knowledge Graph modal/overlay bağlantıları kaldırıldı.
- Knowledge Graph JS component dosyası kaldırıldı.
- Mock Knowledge Graph data dosyası kaldırıldı.
- Knowledge Graph CSS dosyası kaldırıldı.
- `app.js` içindeki Knowledge Graph import, state, veri üretme fonksiyonları ve event listener'ları kaldırıldı.

## Korunanlar
- Mevcut Dashboard yapısı korundu.
- `Sana Özel` carousel yapısı korundu.
- `E-Gazetemi Oku` özelliği korundu.
- PDF/print çıktısı korundu.
- Haber çekme, cache, kategori filtreleri ve mevcut buton mantığı korunmuştur.
